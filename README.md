This repository contains a collection of **terminal-based projects and games developed using C and C++**. The projects demonstrate fundamental programming concepts, including:

* 🧩 Problem Solving
* 🎮 Game Development
* 📂 File Handling
* 🏨 Management Systems
* 🔄 Loops and Conditional Statements
* 🧠 Data Structures and Logic Building

These projects are ideal for:

* 🎓 Students learning C/C++
* 💻 Beginners practicing programming
* 📚 Academic mini-projects
* 🚀 Portfolio demonstrations

# ⚙️ How to Run

## Compile C Programs

```bash
gcc filename.c -o output
./output
```

Example:

```bash
gcc gym.c -o gym
./gym
```

---

## Compile C++ Programs

```bash
g++ filename.cpp -o output
./output
```

Example:

```bash
g++ snack_game.cpp -o snake
./snake
```
